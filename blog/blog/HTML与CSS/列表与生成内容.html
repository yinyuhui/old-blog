列表与生成内容

尹玉慧


列表在我们实现页面布局时也是十分常用的，比如导航条我就比较喜欢用列表做。既然如此重要，当然要好好掌握了~

一、列表项标志类型

１、系统所有 ( list-style-type )

  • disc：　　　　　　　　　　　　实心圆
  • circle：　　 　　　　　　　　　空心圆
  • square：　　　 　　　　　　　 实心方块
  • decimal：　　　　 　　　　　　1，2，3，4 ……
  • decimal-leading-zero：　　　　 01, 02, 03, 04 ……
  • upper-alpha：　 　　　　　　　A， B， C， D ……
  • lower-alpha： 　　　　　　　　 a，b， c， d ……
  • upper-roman：　　　　　　　　大写罗马
  • lower-roman：　　　　　　　　小写罗马
  • none：　　　　　 　　　　　　不使用标志

2、自定义图像 ( list-style-image )

list-style-image: url(xxx.png)


二、列表标志定位 ( list-style-position )

list-style-position: inside | outside ;
  • inside： 列表换行之后第二行与标志平行
  • outside： 列表换行之后第二行与第一行首字平行

三、简写

list-style: url()　square inside;

可以按照任意顺序，当然，如果需要安装顺序写的话，我就不会写上来了

四、插入生成内容 ( content )

要在 p 前后插入内容

p:first |:after {
    content: 要插入的内容;
}

可以是任意元素前后插入内容，此处段落便于书写

插入内容颜色等会继承 p 的设置，如果要单独设置，可以直接在刚刚 content 后面加上 color 语句等

插入内容包括串、 url 等。串使用双引号括起来就会完全相同的插入，即使其中包含了某些标记。若要插入换行符，要用 \A 。仅在书写时换行要使用 \ ，这些都是转义符。

五、计数器

chapter , section , subsec , ordered

1、重置 ( counter-reset )

counter-reset: chapter;         /* 默认重置为0 */
counter-reset: chapter 4;       /* 重置为4 */

2、递增 ( counter-increment )

counter-increment: ordered;         /* 默认为1 */
counter-increment: ordered -2;      /* 递减2 */

对于是使用 chapter 还是 ordered 或是别的，其对应的是要显示计数器的 type， type 是什么， 就写什么

要使用计数器需用以下语句插入

content: counter(ordered);
