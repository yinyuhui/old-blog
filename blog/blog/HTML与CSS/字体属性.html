字体属性

尹玉慧

字体设置是 CSS 中十分重要的，如果不能设置字体，那么我认为 CSS 存在的意义就少了一半。

一、字体类型

字体共有五大类型： serif; 　sans-serif;　momospace;　cursive;　fantasy.

分别对应：有装饰线；　无装饰线；　等宽字体；　模仿手写；　比较有特色的，不能归到上面四类之中。

二、字体设置

  1. 给定一串字体时按优先顺序排列并用逗号隔开；

  2. 如果 a、某个字体含有空格或符号 b、与关键字(如字体系列名)相同，用引号把该字体围起来；

  3. 最后最好放一个通配字体，即上述五大类型；

  4. 如果是类似“微软雅黑”这类字体，最好放中英两种表示方法；

  5. windows 平台和 OS 平台的字体都要设置，兼顾两种平台的字体。

三、字体加粗

font-weight: 值;

上面值可取的有：normal , bold , bolder , lighter , inherit , 100-900

其中 100-900 为九级加粗度，最好不使用，没有统一标准。

四、字体大小

font-size: 值；

上面可取的值有： XX-small ， x-small , small , medium , large , x-large , xx-large , smaller , larger , inherit, 或用像素表示。

一般初始值为medium, 个人偏好直接在body中明确 font-size : medium; (当然也可以设置为其他大小，比如用像素来设置一个具体值)，然后在整个页面中使用 em 控制字体大小，便于整体把控。

五、字体风格(斜体)

font-style: italic;

当然上面还可以取值 oblique 、normal 和 inherit，不过 oblique 是正常文本倾斜放置，看上去没有 italic 美观，所以我对 italic 青睐有加~

六、字体变形

font-variant: small-caps;

这个只能取值 small-caps 和 normal，normal 描述正常文本， small-caps 要求使用小型大写字母文本，通俗的说就是小写变大写，大写变得更大。

<h2>I'm a headline</h2>
<p>Hello, World!</p>

h2{
    font-variant: small-caps;
}

p{
    font-variant: normal;
}

八、使用系统字体

使用系统字体可以使页面看起来跟系统自带的风格相同

有以下取值：

  • caption ( 用于有标题的控件，如按钮 ) ;
  • icon ( 用于对图标加标签 ) ;
  • menu ( 用于菜单，即下拉菜单和菜单列表 ) ;
  • message-box ( 用于对话框 ) ;
  • small-caption ( 用于对小控件加标签 ) ;
  • status-bar ( 用于窗口状态条 ).

例：

botton{
    font: caption;      /* 将按钮字体设置与OS字体完全相同 */
    /* 可以加上语句 font-size: 1em; 重写其字体大小，使与父元素大小相同 */
}



九、字体下载

如果所希望使用的字体大部分用户电脑上并没有，那么可以让其下载字体

格式如下

@font-face { font-family: 字体名，最好使用原始名，也可以自己给名字 ;
    src=url(http://www.xxx.com/......)      /* url跟后面的括号之间一定不要放空格哦 */

接下来在使用到这个字体时，就可以像是用户计算机本地的字体一样

font-family: 上面的字体名;
