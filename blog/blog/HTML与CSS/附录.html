眼看着权威指南就要看完了，过程中写了几篇笔记传到自己的博客上，但这更多是为了填自己学习时留下的坑，留下一些今后忘记了可以查阅的资料，所以可能有很多东西是十分基础的，或者是有很多东西没有被写出来。

有一章是写非屏幕媒体的，我只是大概的过了一遍，并没有深入学习、记忆，这是因为我认为今后使用的可能性非常小，于是没有记笔记.... 现在不用学了也会很快忘记，看了有些印象就好，在今后需要时也容易捡起来

再来补充一点点忽略了或不太熟练的东西，就当作复习了

chip

这个属性用于定义一个剪裁矩形，用于绝对定位元素，只有这个矩形内的内容才可见。超出这个矩形的内容由 overflow 来处理

chip : rect( top, right, bottom, left)

content

这个属性用于在元素之前或之后插入生成内容，用于 :before 和 :after 伪元素

:after{ content: "End~";}
:before{ content: url(xxx.gif);}

font-variant

小型大写字母文本

font-variant: small-caps;

outline

轮廓，在边框外面放置，与边框类似，但无 outline-top 类，只能定义四个边

outline: thin solid red;

overflow

定义如何处理溢出元素内容区的内容

overflow: scroll;           /* 无论是否需要均会产生一个滚动条 */

overflow: hidden;           /* 超出元素框的内容被隐藏 */

overflow: visible;          /* 内容直接完全显示，没有滚动条 */

overflow: auto;             /* 根据需要生成滚动条 */

text-indent

缩进效果，长度值

text-indent: 2em;           /* 可以为负，悬挂缩进的效果 */

text-transform

改变字体大小写，有 uppercase , lowercase , capitalize 这些值选择

text-transform: capitalize;         /* 每个单词首字母大写 */

border-collapse

单元格边框是否重合，默认不重合

border-collapse: collapse;

caption-side

表标题相对于表框的放置位置

caption-side: top | bottom;

子选择器

ul > li { xxx: xxx; }

兄弟选择器

h1 + p { xxx: xxx; }

属性选择器

  • 开始子串　　　　　　　　^=

  • 结束子串　　　　　　　　$=

  • 任意子串　　　　　　　　*=

  • 部分属性　　　　　　　　~=
