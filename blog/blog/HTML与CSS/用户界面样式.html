用户界面样式

尹玉慧


页面样式除了自己设计，还可以模仿系统设计，使其看上去类似于操作系统的一部分。

一、系统字体

共有 6 种系统字体关键字

  • caption: 　　　　　　　 　标题控件使用的字体样式
  • small-caption:　　　　　　标题小控件使用的字体样式
  • icon:　　　　　　　　　　OS图标标签所用的字体样式
  • menu:　　　　　　　　　 下拉菜单和菜单列表中文本所用的字体样式
  • message-box:　　　　　　对话框中文本所用的字体样式
  • status-bar:　　　　　　　 窗口状态栏中文本所用的字体样式

这些值只能用于 font 属性，不能再设置大小、加粗、斜体之类的值，只会保持与用户操作系统设置相同

但其中 icon 是个特例， 如可以在 font: icon; 后加上 font-weight: bold; 语句使字体改为粗体

二、系统颜色

很可能会被一步步废除，最好不使用

body {color: WindowText; background: Window;}

以上语句将为文档指定系统的默认文本颜色和背景颜色

三、光标 ( cursor )

将光标改变为指示手

cursor: pointer; 
cursor: hand;           /* 用于兼容 < IE6 */

将光标改变为文本图标

cursor: text;           /* 光标为 I 形状 */

还可以使用以下值：

  • crosshair
  • move
  • wait
  • progress

定制光标

cursor: url(xxx.cur), pointer;

先加载 url 中的文件，将其作为光标图标，否则使用值 pointer ，可以指定多个 url

四、轮廓

轮廓类似于边框，但不参与到文档流中，也不一定是矩形。 轮廓与边框可以共存，共存时轮廓画在外边框边界外面

轮廓样式 ( outline-style )

与边框样式相同，轮廓样式也有以下几个值可以选择： dotted , dashed , solid , double , groove , ridge , inset , outset , 但没有 hidden 这么个值选择

轮廓宽度 ( outline-width )

thin , medium , thick 以及数值。 与边框一样的欸有没有

轮廓颜色 ( outline-color )

值取颜色值即可，全都与边框一样的~
