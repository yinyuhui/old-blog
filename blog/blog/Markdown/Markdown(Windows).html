这是我首次使用Markdown，必须记录下来啊，毕竟这标志着我又向程序媛靠近了一些，也为不知如何入门的提供些方法。这篇文就是使用MarkdownPad写的哦~

入坑原因

在不小心跳进前端之后，就陆续接触到了 github 、Markdown 等，初次认识它时仅仅知道这是一个文本编辑器一样的东西，一般都是用来写博客或者记笔记之类的。不过那时还是一个小白，对此嗤之以鼻，认为博客有那么多平台可以直接套模板，笔记我有记事本和 word ，有这两个就够了， markdown 是什么，压根都没有听说过。

随着前端学的越来越深入，笔记越记越多，word 根本 hold 不住我对格式的要求，更不要说记事本了。这时我意识到一个好的笔记工具是十分重要的，于是开始了解 markdown ，突然之间就开启了新世界的大门。

在finish后，启动时会出现一个注册页面（安装的时候没有截图欸，描述不知道清不清楚），点击有单词 " KEY " 的按钮，接下来请看MarkdownPad注册码，提供了邮箱和密钥，亲测可用。
